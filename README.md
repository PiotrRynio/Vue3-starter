## Overview

**RickyAndMorphyApp** is a frontend application created in Vue 3 framework.

**RaM API URL:** [https://rickandmortyapi.com/documentation/#graphql](https://rickandmortyapi.com/documentation/#graphql)

###### The following technologies/library/methodology were used:

- Vue.js 3 framework
- Vue 3 Composition API
- Vue router
- Axios
- Atomic design architecture
- Prettier
- Husky
- UUID generator
- SCSS
- BEM methodology
- Mobile First & and full responsive
- Clean Code
- the newest features of ECMAScript
- Figma project
- Pixel-perfect implementation
- Git & GitHub
- IntelliJ

## Running the project

Running this project locally

###### From the repo:

1. Clone this project locally.
2. Run `npm install` in your bash/command line.
3. Run `npm run serve` in your bash/command line.
4. Go to `http://localhost:8080/` in your browser (or other from your bash/command line info).
